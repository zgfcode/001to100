第一章：JavaScript简介

JavaScript于1995诞生
ECMA-262规定了JavaScript的语法，类型，语句，关键字，保留字，操作符，对象
ECMAScript的宿主环境：Web浏览器，Node，Adobe Flash
JavaScript、Adobe ActionScript实现了ECMAScript

EMCAScript是一种专为与网页交互而设计的脚本语言，由下列是哪个不同的部分组成：
ECMAScript，由ECMA-262定义，提供核心语言功能
文档对象模型（DOM），提供访问和操作网页内容的方法和接口
浏览器对象模型（BOM），提供与浏览器交互的方法和接口

3.1 语法

3.1.1 区分大小写
一切（变量、函数名和操作符）都区分大小写
3.1.2 标识符
：变量、函数、属性的名字，或者函数的参数
第一个字符必须是一个字母、下划线或一个美元符号
其他字符可以是字母、下划线、美元符号或数字
惯例下，ECMAScript标识符采用驼峰大小写格式，最佳实践
3.1.3 注释
C风格的注释：单行注释，块级注释
3.1.4 严格模式（strict mode）
"use strict": 在顶部添加，在函数内部的上方添加
此模式下，ECMAScript中一些不确定的行为将得到处理，某些不安全的额操作会抛出异常
3.1.5 语句
语句以分号结尾，若省略，由解析器确定语句的结尾


3.2 关键字和保留字

关键字：

保留字：

class const enum export extends import super
严格模式下：
implements interface let package private protected public static yield
第5版下，关键字和保留字不能作为标识符使用，但现在可以用作对象的属性名

3.3 变量

ECMAScript的变量是松散类型的，可以用来保存任何类型的数据，每个变量仅是一个用于保存值得占位符，定义变量时要使用var操作符。
var message;
未经过初始化的变量，会保存一个特殊的值————undefined
var message='hi';
这样初始化变量并不会包它标记为字符串类型，仅是赋值，可以在修改变量值的同时修改值的类型

用var操作符定义的变量将成为定义该变量的作用域中的局部变量
省略var操作符，可以创建一个全局变量

可以使用一条语句定义多个变量，每个变量用逗号隔开


3.4 数据类型

ECMAScript中有5种简单数据类型（基本数据类型）：
Undifined Null Boolean Number String
1种复杂数据类型————Object, 本质是由一组无需的名值对组成的
ECMAScript不支持任何创建自定义类型的机制，所有值最终都是上述6种类型之一。
数据类型具有动态性

3.4.1 typeof操作符
检测给定变量的数据类型
typeof 一个值，返回：
undefined: 如果这个值未定义

string

function：这个值是函数

typeof操作符的操作数可以是变量，也可以是数值字面量

3.4.2 Undefined类型
使用var声明变量但未对其加以初始化时，变量的值为undefined

3.4.3 NULL类型
null值表示一个空对象指针，操作符检测返回"object"
如果定义的变量准备在将来用于保存对象，那么最好将该变量初始化为null
位于null和undefined之间的相等操作符（==）总会返回true

3.4.4 Boolean类型
使用得最多的类型，只有两个字面值：true和false
ture不一定等于1，false不一定等于0，区分大小写
ECMAScript中所有类型的值都有雨这两个Boolean值等价得到值
转型函数Boolean();
空字符串、0、NaN、null、undefined转换为false
（n/a（或N/A），是not applicable的缩写，意为"不适用"。

3.4.5 Number类型
进制类型，浮点数值，科学计数法，数值范围
NaN，即非数值（not a number）是一个特殊的数值
用于表示一个本来要返回数值的操作数未返回数值的情况
任何涉及NaN的操作都会返回NaN
NaN与任何值都不相等
isNaN()函数，不是数值并且不能转换为数值，返回true
##数值转换：三个函数可以把非数值转换为数值：
Number(), parseInt(), parseFloat()
第一个可以用于任何数据类型
另两个专门用于把字符串转换成数值
Number()函数的转换规则：null返回0，undefined返回NaN，空字符串返回0

3.4.6 String类型
字符串可以由双引号或单引号表示
字符串一旦创建，它们的值就不能改变
把一个值转换为一个字符串有两种方式：1. toString()方法，null和undefined没有这个方法
调用toString()方法通常不必传递参数，调用数值的toString()方法，可以传递一个参数：输出数值基数
2. 转型函数String()，在不知道要转换的值是不是null或undefined的时候使用

3.6 语句（流控制语句）
if语句
	if (condition) statement1 else statement2
	condition(条年)可以是任意表达式，对这个表达式求值得结果不一定是布尔值
	ES会自动调用Boolean()转换函数将表达式的结果转换为布尔值
	条件的求值结果决定是执行语句1还是语句2，语句可以是一行代码或一个代码块


do-while语句（一种后测试循环语句）
	do {
		statement
	} while (expression);
	在循环体的代码执行后，再测试出口条件；在对表达式求值之前，循环体内的代码至少会被执行一次

while语句(前测试循环语句)
	while (expression) statement
	循环体内的代码被执行之前，就会对出口条件求值。循环体内的条件有可能永远不会被执行

for语句（前测试循环语句）
	for (initialization; expression; post-loop-expression) statement
	具有在执行循环之前初始化变量和定义循环后要执行的代码的能力。
	使用while循环做不到的，使用for循环同样也做不到。for循环只是把与循环有关的代码集中放在了一个位置。
	for循环的变量初始化表达式中，可以不适用var关键字，该变量的初始化可以在外部执行。

	##ES中不存在块级作用域，在循环体中定义的变量可以刻在外部访问到。

for-in语句（一种精准的迭代语句，可用来枚举对象的属性）
	for (property in expression) statement

label语句
	label: statement
	在代码中添加标签，以便将来使用

break和continue语句
	用于在循环体中精确地控制代码的执行。
		break语句会立即退出循环，强制继续执行循环后面的语句
		continue语句会立即退出循环，但退出循环后会从循环的顶部继续执行。
